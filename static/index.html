§1

Transmitter Protocol

Cover acquisition → preprocessing → feature audit → embedding → encryption → dispatch.

I

Cover Acquisition

Select a cover image. Its pixel field becomes the carrier for the hidden payload.

II

Preprocessing

Grayscale conversion and histogram equalization, as in the C++ recognizer front-end (rtsig.cpp).

III

Local-Feature Audit

Scale-invariant keypoint extraction over the cover (the desktop prototype's SURF stage).

IV

LSB Embedding

The secret message is written bit-by-bit into pixel least-significant bits: a 4-byte length header, then the payload — 8 carrier bytes per hidden byte.

V

Encryption & Dispatch

The stego image is RC4-encrypted under a biometric feature-vector key; the key itself is AES-wrapped (SHA-1 → AES/ECB) before transmission.
